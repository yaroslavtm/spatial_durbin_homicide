# Модель пространственной авторегрессии Дурбина
Выявление факторов, определяющих уровень умышленных убийств в штатах США при помощи модели пространственной авторегрессии (Spatial Durbin Model) с двусторонними фиксированными эффектами и коррекцией смещения Ли-Ю (Lee, Yu; 2010).  

Зависимая переменная: число умышленных убийств на 100 000 человек (FBI UCR).  

Регрессоры: реальный доход на душу населения, потребление этанола, уровень безработицы, смертность от передозировки опиоидами, прокси-показатель владения огнестрельным оружием (FS/S), индекс Джини, уровень инкарцерации.  

Основные результаты: коэффициент пространственной авторегрессии ρ = 0,573 (p <0,001), R² = 0,936. Значимые суммарные эффекты: индекс Джини (+0,243), потребление этанола (+4,208), безработица (-0,613).  

# Структура репозитория
01_data_and_W.R — загрузка данных, построение матрицы пространственных весов, описательная статистика и корреляционный анализ.  

02_spatial_models.R — Pooled OLS, Two-Way FE, тест Хаусмана, Moran's I, LM-тесты, SDM, декомпозиция эффектов (Monte Carlo, 10 000 симуляций), LR-тест SDM vs SAR.  

README.md — описание репозитория и инструкции для выполнения скриптов.

data.xlsx — данные, используемые при выполнении скрипта 01_data_and_W.R.

# Источники данных

| Переменная | Источник |
|---|---|
| Умышленные убийства | FBI UCR |
| Доход, население | BEA, SAINC1 |
| Безработица | USDA ERS |
| Потребление этанола | NIAAA via openICPSR |
| Смертность от опиоидов | CDC WONDER |
| Прокси-показатель владения оружием (FS/S) | CDC WISQARS |
| Индекс Джини | Census Bureau ACS |
| Уровень инкарцерации | BJS |
| Шейпфайл штатов | Census Bureau |

# Запуск
1. Скрипты используют единый файл `data.xlsx`, 
   сформированный вручную из источников, перечисленных выше. 
   Методология сбора и агрегации данных описана в Главе 1 курсовой работы.
2. Скачать шейпфайл `cb_2018_us_state_500k` с сайта Census Bureau.
3. Указать пути к файлам данных и шейпфайлу в скриптах.
4. Запустить 01_data_and_W.R — создаст workspace_01.RData. Указать в скрипте 02_spatial_models.R путь к workspace_01.RData.
5. Запустить 02_spatial_models.R — выведет результаты всех моделей и тестов.
